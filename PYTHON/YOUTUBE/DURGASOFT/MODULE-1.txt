If you type the datatype, then it is STATICALLY TYPED PROGRAMMING - C & JAVA examples

STATICALLY TYPED EXAMPLE: (C program)
#include <stdio.h>
void main()
{
int a,b;
a=10;
b=20;
print(a+b);	--->	30
}



DYNAMICALLY TYPED EXAMPLE: (Python program)
a,b=10,20
print(a+b)	--->	30

a,b,c,d,e=10,20,30,40,50
type(a)		--->	<class 'int'>

a=True
type(a)		--->	<class 'bool'>
